### 查看当前发行版本
```java
lsb_release -a
uname -a
```
查找/etc文件
```java
对于Debian 及其衍生版： cat \etc\issue cat \etc\issue.net cat \etc\lsb-release cat \etc\os-release

对于CentOS： cat \etc\centos-release 对于Fedora： cat \etc\fedora-release
```
